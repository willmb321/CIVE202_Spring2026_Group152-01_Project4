# CIVE202_Spring2026_Group152-01_Project4
This project assesses the risk of natural disasters using the data from the National Risk Index (NRI), which is provided by the Federal Emergency Management Agency (FEMA). It also evaluates the risk through a model that was created by the researchers themselves. By making use of the tract-level data in two selected states, we were able to clean, merge, and conduct a sensitivity analysis on the data provided by the NRI and CDC Social Vulnerability Index. We used Python and GeoPandas in doing so and produced summaries and maps of the data sets.
